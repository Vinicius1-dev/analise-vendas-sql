# 📊 Análise de Vendas com SQL

Projeto de análise de dados de vendas utilizando SQL.

## 🎯 Objetivo
Analisar dados de vendas e gerar insights importantes.

## 📊 O que foi analisado
- Total de vendas
- Produto mais vendido
- Cliente que mais comprou
- Média das vendas

## 🛠️ Tecnologias
- SQL

## 📚 Aprendizados
- SELECT
- GROUP BY
- SUM, AVG
- JOIN
